# loan-approval-prediction
Machine Learning model to predict loan approval
#  Bank Loan Approval Prediction

##  Project Overview

This project predicts whether a loan application will be approved based on applicant details like income per annum, cibil score, loan amount, etc.

##  Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

##  Steps Performed

* Data Cleaning (handled missing values)
* Exploratory Data Analysis (EDA)
* Label Encoding
* Model Building (Logistic Regression, Random Forest, SVM, KNN)
* Model Evaluation

##  Performance

* Accuracy: Highest for Random Forest Algorithm

## Visualizations

* Bar Plot
* Heatmap

##  How to Run

1. Install libraries:
   pip install pandas numpy scikit-learn matplotlib seaborn

2. Run:
   loan_approval_dataset.ipynb
